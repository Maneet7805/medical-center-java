# medical-center-java
A GUI-based Java application for managing patient records, appointments, and prescriptions. Built using Swing and AWT with OOP principles.  
OOP concepts includs encapsulation, inheritance, polymorphism, and abstraction.

## Technologies
- Java
- Swing & AWT

## Features
- Add and manage patient records
- Schedule and track appointments
- Manage prescriptions and medications
- GUI-based interface for ease of use

## How to Run
1. Ensure Java is installed.
2. Open the project in any Java IDE.
3. Compile and run `Main.java`.

## Notes
- Designed for desktop use.
- Demonstrates object-oriented programming principles.
